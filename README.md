# Portable Office Cabin

Premium Portable Cabins & Prefab Solutions in India

## About

Leading manufacturer of portable cabins, container offices, prefab homes, and site office solutions. Quality structures delivered ready for use.

## Technology Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend)
- Framer Motion

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Build

```bash
npm run build
```

## Deployment

The application is deployed on Digital Ocean App Platform with custom domain `portableofficecabin.com`.
