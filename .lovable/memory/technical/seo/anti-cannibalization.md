# SEO Anti-Cannibalization Master Instruction

## Primary Objective
Never intentionally create keyword cannibalization. Every page must have a unique SEO purpose, unique target keyword, and unique search intent.

## Keyword Assignment
- One Primary Keyword = One URL Only.
- Once assigned, no other page may target the same keyword.
- Before creating any new page, check existing URLs, titles, H1s, and keywords.
- If keyword exists, create a supporting page with a different keyword + intent.

## Product Pages — must each have unique:
Primary keyword, SEO title, H1, meta description, content, FAQ, schema markup, images, image alt text. Never copy from other product pages.

## Project Pages
- Must NOT target product keywords.
- Allowed: location/use-case framing (e.g., "20ft Office Cabin Installed at Construction Site", "Site Office Cabin Delivered in Bangalore").
- Must internally link to the relevant product page; never compete with it.

## Blog Pages
- Informational keywords only (cost guide, dimensions explained, comparisons).
- NOT allowed: commercial keywords like "manufacturer", "buy", "best supplier" — those belong to product pages only.

## Location Pages
- Unique content, local project references, local use cases, local FAQs, local delivery info.
- Never duplicate content across city pages.

## Internal Linking Hierarchy
Home → Category → Product → Project → Blog.
Projects and blogs must link up to the relevant product page. Product pages remain authority for commercial keywords.

## URL Rules
Every page: unique slug, unique keyword focus, unique title, unique H1. No similar URLs targeting the same keyword.

## Canonical Rules
If similar content exists, canonical to the primary page. Avoid indexable duplicates.

## Content Quality
100% original, human-readable, AI-detection optimized, plagiarism-free, written specifically for the assigned keyword, different from every existing page.

## Mandatory Pre-Publish Validation
Before creating/publishing ANY page, check:
1. Keyword already exists?
2. Similar page exists?
3. Title tag conflict?
4. H1 conflict?
5. URL conflict?
6. Search intent conflict?
7. Internal linking structure intact?

If ANY conflict → STOP. Recommend a unique keyword strategy instead.

## Final Rule
Maintain a Keyword Mapping System: every primary keyword permanently assigned to one dedicated URL. No new page may target the same primary keyword, title intent, or search intent as an existing page.
